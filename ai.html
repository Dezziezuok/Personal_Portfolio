AI/ML Engineering Projects

Project 1: AI Virtual Painter (OpenCV, Python)

Developed a gesture-based virtual painting application with real-time hand tracking. Users draw directly in the air, selecting from 7+ colors and an eraser tool via finger gestures.
Implemented multiple brush modes, canvas management, and creative filters. Key features:
Hand gesture recognition to switch between brushes, clear canvas, and save artwork without keyboard/mouse.
Drawing tools: normal brush, pixel brush, eraser, and customizable thickness.
UI integration with dynamic header bar for color selection and visual feedback.
Image processing & style transfer with OpenCV to apply cartoon and oil painting effects.
Automated saving system for storing completed drawings in PNG format.

Project 2: Drawing Game (OpenCV, Python)

Built an interactive gesture-based drawing game using Python and OpenCV. Players receive randomized word prompts (e.g., “cat,” “robot,” “pizza”) and draw them in real time with hand tracking.
Implemented a 40-second timer, drawing via fingertip detection, and automatic saving of completed sketches. Key features:
Integrated hand tracking module for gesture detection and fingertip drawing input.
Designed a game loop with randomized prompts, countdown timer, and on-screen feedback.
Applied image processing techniques (masking, overlay blending, thresholding) to merge drawings with webcam feed.
Automated storage of player creations by saving canvas output as PNG images.

Project 3: Recommendation System (MovieLens, Python)

Built a movie recommendation engine using collaborative filtering and matrix factorization (SVD), achieving 20% lower RMSE than baseline popularity ranking on MovieLens dataset.
Implemented content-based filtering with TF-IDF on movie genres to personalize results.
