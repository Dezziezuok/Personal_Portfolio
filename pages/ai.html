Project 1: AI Virtual Painter (OpenCV, Mediapipe, Python)

Developed a gesture-based virtual painting application with real-time hand tracking. Users draw directly in the air, selecting from 7 colors and an eraser tool via finger gestures.
Implemented multiple brush modes, canvas management, and creative filters.
UI integration with a dynamic header bar for color selection and visual feedback. Image processing & style transfer with OpenCV to apply visual effects. Automated saving system for storing completed drawings in .png format.

Project 2: Book Recommendation System (Gradio, Hugging Face, PyCharm, Goodreads Kaggle Dataset)

Developed an interactive book recommendation system leveraging machine learning and natural language processing to provide personalized suggestions.
Processed Kaggle-sourced data through custom cleaning pipelines for optimal model performance.
Encoded 5,600+ book descriptions into semantic embeddings using pre-trained LLMs, enabling content-aware similarity search.
Developed a hybrid recommendation model integrating semantic similarity, metadata (genre, author, fiction/non-fiction), and emotion & sentiment analysis to enable contextual and tone-based book suggestions.
Designed a scalable architecture using vector databases (Chroma) for real-time inference across thousands of books.
Replaced Books without a thumbnail with a placeholder image for better UI consistency.
Built an interactive UI with Gradio, enabling users to input descriptions or preferences and receive instant recommendations.
Integrated the OpenAI API via .env for semantic search and classification.
Stack: Python 3.11, Pandas, Matplotlib, Transformers, LangChain, Chroma, Hugging Face, Gradio, Jupyter.

Project 3: Drawing Prompt Generator (OpenCV, Mediapipe, Python)

Built an interactive gesture-based drawing game using Python, OpenCV and Mediapipe:
Integrated hand tracking module for gesture detection and fingertip drawing input.
Designed a game with randomized prompts (cat, robot, pizza, etc), with a 30s countdown timer and player(s) draw them in real time.
Applied image processing techniques to merge drawings with webcam feed.
Automated storage of player creations by saving canvas output as .png images.
